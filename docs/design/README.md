# Enclave — Design Contract (C2)

The approved visual contract for Enclave's surfaces. Any visual — the C2 Command
Center, the board UI, launch assets — reconciles to this.

- **`c2-mockups-v1.html`** — Command Deck + boards + the gate-takeover motif.
- **`c2-mockups-v2.html`** — production target (boot → initiation → deck).

## The contract
- **Palette:** amber `#E8A33B` on obsidian `#0A0B0D`.
- **Type:** Space Grotesk (display) + IBM Plex Mono (data/labels).
- **Motif:** the hex-seal; tasks read **PROVEN** / **REJECTED**; a full-screen
  **Gate Takeover** on rejection.
- **Sovereignty spectrum** is first-class: AIR-GAP / HYBRID / GRID (the
  `SOVEREIGNTY MODE` dial) — kept verbatim; it is the product, not brand chrome.

## Provenance
Reconciled from the approved C2 mockups: the `SOVEREIGN AI FACTORY` wordmark was
brought in line with the public brand — **Enclave** (D1). Only the brand wordmark
changed (title, boot `b1`, sidebar `wm-1/wm-2`); the `SOVEREIGNTY MODE` feature and
all layout/interaction are untouched. Verified rendered under the Enclave wordmark
at 1440×900 (both files).

> Building the C2 Command Center is **out of scope** for the launch mission — it is
> the next mission. These files are the design contract only.
